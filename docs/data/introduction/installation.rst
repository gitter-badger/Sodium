Installation
============

Sodium can be installed through composer,

command line
------------

.. code-block:: php
   :linenos:

   composer require lokesh/sodium

composer.json
-------------

.. code-block:: json
  :linenos:

  "require":{
   "lokesh/sodium":"dev-master"
  }
